# Calculadora Funcional

Calculadora web desarrollada con React que permite realizar operaciones matemáticas básicas. 
Podes integrarla en tu web de manera totalmente responsiva.

##  Características

- Operaciones básicas (suma, resta, multiplicación, división)
- Soporte para números decimales
- Interfaz moderna y responsive
- Diseño inspirado en calculadoras modernas

## Tecnologías

- React
- CSS3
- JavaScript (ES6+)

## Instalación

\`\`\`bash
npm install
npm start
\`\`\`
